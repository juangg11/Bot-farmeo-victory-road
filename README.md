# Bot-farmeo-victory-road
Bot automático para farmear gachapones en Inazuma Eleven Victory Road.

# Instalación

En la página principal del repositorio, busca el botón verde "Code" (Código)
Haz clic en él y selecciona "Download ZIP" (Descargar ZIP)
Guarda el archivo ZIP en tu ordenador
Localiza el archivo ZIP descargado en tu carpeta de descargas
Haz clic derecho sobre el archivo ZIP
Selecciona "Extraer todo..." o "Extract here"
Se creará una carpeta con los archivos del bot


# Uso

Abre Inazuma Eleven Victory Road y ve hasta el modo BB
IMPORTANTE: Deja el juego en esta pantalla del menú (no inicies la partida)
Abre la carpeta del bot extraída anteriormente
Busca el archivo ejecutable (.exe)
Haz doble clic para ejecutarlo
El bot comenzará a funcionar automáticamente


# Importante

NO toques el ratón ni el teclado mientras el bot está funcionando
El bot farmeará gachapones de forma automática rindiéndose en la segunda parte para ir más rápido


# Detener el Bot
Para detener el bot pulsa la tecla F8
